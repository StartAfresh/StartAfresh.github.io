因为要部署到gihthub上，我们需要准备node.js包管理工具，以及git将代码推到github进行管理

git进入网站下载

2.安装node

node.js进入网站下载

注意事项

hexo + Typora 图片上传问题

1. hexo + Typora 图片上传问题

解决步骤：

  1. 修改typora图片放置的位置。当自己新建一个文章时，添加图片自动创建存放图片的文件（文件夹的名字与文章笔记相同）

  1. 修改博客根目录下的站点配置（_config.yml）文件中的post_asset_folder字段设置为true
    • 当设置 post_asset_folder 参数后，在hexo n命令建立文件时， 会自动建立一个与文章同名的文件夹，把与该文章相关的所有图片资源都放到此文件夹内，这样就可以方便的使用图片资源
    • 同时，只有当post_asset_folder设置为true后，后续安装的插件才会起作用

​

  1. 安装插件

    • 到根目录下 npm install https://github.com/CodeFalling/hexo-asset-image –save命令来进行插件的安装

    • 当文章全部写完后，使用Typora的替换功能（替换功能包含删除功能，当替换的内容什么都不输入时为全部删除）将所有图片地址前面多余的部分删除即可
